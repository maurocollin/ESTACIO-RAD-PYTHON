# Desenvolvimento Rápido de Aplicações em Python 🐍
> Repositório destinado aos exercícios e projetos da disciplina ARA0095 - Estácio de Sá.

Este repositório contém a implementação de soluções focadas em **Metodologia RAD**, manipulação de dados, tratamento de exceções e persistência de arquivos.

## 🚀 Tecnologias Utilizadas
* **Python 3.x**
* **Google Colab** (Prototipagem rápida)
* **GitHub** (Versionamento)

## 📁 Estrutura do Repositório
* `Tema_3/`: Exercícios de Manipulação de Dados e Exceções.
    * `calculadora_divisao.ipynb`: Tratamento de erros matemáticos e de entrada.
    * `calculo_fatorial.ipynb`: Lógica de repetição com validação de tipos.
    * `contador_palavras.ipynb`: Manipulação de arquivos de texto (.txt).
    * `formatador_nomes.ipynb`: String manipulation e persistência de dados.

## 🛠️ Como Executar os Exercícios
1. Acesse a pasta do exercício desejado.
2. Clique no botão **"Open in Colab"** (presente no topo dos notebooks).
3. Execute as células para ver o código em funcionamento.

## 📝 Boas Práticas Aplicadas
* **Tratamento de Exceções:** Uso de blocos `try-except-finally` para evitar crash do sistema.
* **Manipulação Segura de Arquivos:** Uso do gerenciador de contexto `with` para garantir o fechamento de buffers.
* **Padronização PEP 8:** Código escrito seguindo as convenções de estilo da comunidade Python.
